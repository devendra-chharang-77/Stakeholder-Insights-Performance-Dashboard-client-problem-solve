# Stakeholder-Insights-Performance-Dashboard-client-problem-solve
📊 Project Overview
An interactive Tableau dashboard designed for insurance stakeholders to analyze premium trends, insurer performance, profitability, and asset distribution.
🔍 Business Problem
The client lacked a centralized reporting solution to monitor important business KPIs and stakeholder insights efficiently.

✅ Solution
Built a fully interactive Tableau dashboard with dynamic filtering, KPI tracking, cross-chart interactivity, and export functionality.
🚀 Features
Premium Trend Analysis
Insurer Performance Comparison
Profitability Analysis
AUM Distribution
Interactive Filters
Dynamic KPIs
PDF Export
Role-based Access
🛠 Tools Used
Tableau Public
Excel / CSV
Data Visualization
Business Intelligence
